TicTacToe.React
===============

A TicTacToe game built using React.js

Install Instructions
===============

Prerequisites:
* node.js
* npm

Install bower:
`npm install -g bower`


Clone repo and open shell in directory

`npm install`

`bower install`


Build app by invoking gulp: 
`gulp`


To run web server (simple node file server):
`node server.js`

Finally, connect to `localhost:8080`
